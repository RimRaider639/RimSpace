# RimSpace
 ```v 1.0.0```
 
Social media app using react native. User can create an account, login, post pictures, comment and like.

[Download APK for Android](https://drive.google.com/uc?export=download&id=1z4MLlZvzkO2Q8h5F3TmSfru-FvRKMueB)
